GFORTH

I wrote this FORTH program to be used as a replacement for the failed ACCII climate control system in my 1980 300TD. The idea put simply was to replace the actual unit with an esp32 running espForth, replace the servo with an electronic valve, and wire the esp into the vac solenoids that control vents with MOSFETS.

This has all the fucntionality of the original besides automatic control, which I had set up to use two temp sensors. One under the hood for OAT and one in the cabin for IAT with the eventual idea being i could also use humidity sensors for intelligent defrost. 

This is a representation of the logic without real pin assignments. A self test function is included.

Unfortunately I had to sell the car but I am posting this in case anyone else is interested.
